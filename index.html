Referência Falando um pouco sobre o HTML O HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar e apresentar o conteúdo de páginas da web. Ele é a base fundamental para a criação de sites e aplicações web. O HTML utiliza tags (ou elementos) para definir a estrutura e o significado dos diferentes elementos presentes em uma página web. Cada tag é envolvida por símbolos de menor (<) e maior (>) e pode conter atributos que fornecem informações adicionais sobre o elemento. Um documento HTML básico geralmente começa com a declaração , seguida pela tag , que envolve todo o conteúdo da página. Dentro da tag , temos as tags e . A tag contém informações sobre a página, como o título exibido na barra do navegador, a inclusão de folhas de estilo CSS e scripts JavaScript. A tag contém o conteúdo visível da página, como texto, imagens, vídeos e outros elementos.

O surgimento do HTML

O HTML foi criado por Tim Berners-Lee no início dos anos 1990, enquanto ele trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) em Genebra, na Suíça. Berners-Lee estava desenvolvendo um sistema de gerenciamento de informações que permitiria aos cientistas compartilhar e acessar facilmente documentos e dados entre si.

Em 1990, ele propôs o conceito de um sistema hipertextual que permitiria aos usuários navegar por documentos interconectados, conhecido como WorldWideWeb. Para tornar isso possível, Berners-Lee precisava de uma linguagem de marcação que permitisse a estruturação de documentos hipertextuais.

Assim, em paralelo ao desenvolvimento do sistema WorldWideWeb, Berners-Lee criou o HTML como uma linguagem simples para marcar a estrutura e o conteúdo desses documentos.

O que são as famosas TAG's

Basicamente as tags são elementos fundamentais para estruturar e organizar o conteúdo de uma página da web. As tags são representadas entre "<>" e "", e existem várias delas disponíveis para diferentes finalidades.

Aqui estão algumas das tags mais comuns no HTML:

  1. Html e body: Essas tags são as estruturas básicas de uma página HTML. A tag "html" envolve todo o conteúdo da página, enquanto a tag "body" contém o conteúdo visível para os usuários.
  2. Head: A tag "head" é usada para fornecer informações sobre a página, como o título exibido na barra de título do navegador, a inclusão de folhas de estilo CSS, scripts JavaScript e outras metainformações.
  3. H1 e h6: Essas tags são usadas para definir cabeçalhos ou títulos de diferentes níveis. O "h1" é o cabeçalho mais importante, geralmente usado para o título principal da página, enquanto o "h6" é usado para subtítulos de menor importância.
  4. p: A tag "p" é usada para criar parágrafos de texto. É uma das tags mais comuns para a formatação de conteúdo textual.
  5. "a": A tag "a" é usada para criar links em uma página HTML. Ela requer um atributo "href" para especificar o destino do link.
  6. "img": Essa tag é usada para exibir imagens na página. Ela requer um atributo "src" que especifica o caminho da imagem.
  7. "ul" e "li": Essas tags são usadas para criar listas não ordenadas. A tag "ul" envolve a lista e a tag "li" é usada para cada item da lista.
  8. "div" e "span": Essas tags são usadas para agrupar e estilizar elementos em bloco (div) ou em linha (span). Elas são frequentemente usadas em conjunto com o CSS para aplicar estilos personalizados.

Essas são apenas algumas das muitas tags disponíveis no HTML. Cada uma delas desempenha um papel importante na estruturação e formatação do conteúdo de uma página da web, permitindo que os desenvolvedores criem uma experiência visualmente agradável e organizada para os usuários.
